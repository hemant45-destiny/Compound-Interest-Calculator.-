# Compound Interest Calculator (Python)

## 📌 Project Description

This project is a simple **Compound Interest Calculator** written in Python.
The program takes the **principal amount, interest rate, and time** as input from the user and calculates the **total amount after compound interest**.

The program also validates user input using **while loops** to ensure that the values entered are not zero or negative.

---

## ⚙️ Technologies Used

* Python 3
* While Loops
* Input Validation
* Mathematical calculations
* Format specifiers (f-strings)

---

## 🚀 How It Works

1. The user enters the **principal amount**.
2. The user enters the **interest rate**.
3. The user enters the **time in years**.
4. The program checks that all values are **greater than zero** using while loops.
5. The compound interest formula is applied.
6. The total amount is displayed.

---

## 📐 Formula Used

Compound Interest Formula:

A = P (1 + r/100)^t

Where:

* **P** = Principal amount
* **r** = Interest rate
* **t** = Time in years
* **A** = Total amount after interest
